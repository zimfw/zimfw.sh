Getting Started

  • First, make sure you've got zsh installed.
  • Next, install Zim, using either a one-liner or manual installation.
  • Once installed, check out our Themes and Modules to customize your experience.
  • Take a look at the cheatsheet to see all the commands and aliases Zim offers.
